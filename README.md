# simple_push_swap_tester

Quick one-liner setup — run this from the root of your `push_swap` project:

```bash
git clone https://github.com/yubinex/simple_push_swap_tester.git && mv simple_push_swap_tester/autotest.sh simple_push_swap_tester/walkthrough.sh . && rm -rf simple_push_swap_tester && chmod +x autotest.sh walkthrough.sh
```

Then run the tests:

```bash
./autotest.sh
```
```bash
./walkthrough.sh
```
